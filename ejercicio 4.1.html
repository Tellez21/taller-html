ETIQUETAS DE HTML

  • HTML (Hypertext Markup Language) - Lenguaje de marcado de hipertexto utilizado para crear páginas web.
  • Tag (etiqueta) - Una palabra o frase rodeada de corchetes angulares que indica la estructura o el estilo de un elemento en una página web.
  • Elemento - Un componente de una página web, como un título, un párrafo o una imagen.
  • Atributo - Un valor asociado con una etiqueta o elemento que proporciona información adicional.
  • Valor - El contenido de un atributo o elemento.
  • CSS (Cascading Style Sheets) - Un lenguaje de estilo utilizado para controlar la presentación de una página web.
  • Script - Un fragmento de código que se ejecuta en el lado del cliente o del servidor.
  • URL (Uniform Resource Locator) - Una cadena de caracteres que identifica la ubicación de un recurso en internet.
  • Enlace (link) - Una conexión a otra página web o a un recurso en la misma página.
  • Imagen - Un elemento visual que se puede incorporar a una página web.
